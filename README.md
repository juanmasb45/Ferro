# Ferro 🚆

Una aplicación de Android desarrollada con **Android Studio** en **Arch Linux**.

## 🛠️ Tecnologías
* **Lenguaje:** Kotlin / Java (dependiendo de tu elección)
* **IDE:** Android Studio
* **SO de desarrollo:** Arch Linux 🐧
* **Build System:** Gradle

## 🚀 Instalación
1. Clona el repositorio:
   ```bash
   git clone https://github.com/juanmasb45/Ferro.git
   ```
2. Abre el proyecto en Android Studio.
3. Sincroniza Gradle y ejecuta en un emulador o dispositivo físico.
